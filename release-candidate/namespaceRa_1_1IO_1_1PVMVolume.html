Loads a volume in pvm format. See The Volume Library at http://schorsch.efi.fh-nuernberg.de/data/volume/.

Parameters
filenamename of the file to load
widthwidth of the loaded data
heightheight of the loaded data
depthdepth of the loaded data
bytesnumber of byte per density value
scalexx scale of the voxel
scaleyy scale of the voxel
scalezz scale of the voxel
descText describing the data
courtesyText with copyright info
parameterText with aquisition parameters description
commentText comenting the data
Returns
a pointer to the raw density data, nullptr if the file was not loaded.
